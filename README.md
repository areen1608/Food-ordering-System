# Food Ordering Project

![image](https://github.com/eminkmru/food-ordering-with-NextJS/assets/89755021/272ac072-0702-4c83-9a79-a09a2eb43b4c)


Hello,

I worked on the Food Ordering Project, which is a comprehensive web application developed using various technologies.

## Project Content and Features

-Developed an SPA architecture.
-Implemented UI with React and Next.js framework.
-Styled and added transitions using Tailwind CSS.
-Enhanced state management through Redux Toolkit.
-Enabled secure authentication via Next-Auth.
-Managed forms and validations using Formik & Yup.
-Connected and managed data using MongoDB.
-Built Admin Dashboard and user profile system.
## Extra Features Developed by Me

In addition to the core features taught by the course instructors, I added some extra features to enrich the project:

-Improved component data flow with Redux Toolkit.
-Built an intuitive error handling system to guide users.
-Refined MongoDB queries to increase efficiency.
-Introduced customizable order functionality for users.
